Azure Service

Azure มีบริการคอมพิวเตอร์คลาวด์ 3 รูปแบบหลัก คือ Infrastructure as a Service (IaaS), Platform as a Service (PaaS) และ Software as a Service (SaaS)

Azure IaaS ให้บริการทรัพยากรคอมพิวเตอร์เสมือนจริง เช่น เซิร์ฟเวอร์ พื้นที่จัดเก็บข้อมูล และเครือข่าย เพื่อช่วยให้ผู้ใช้สามารถสร้างและจัดการ virtual machine และ application ของตนเองในคลาวด์ได้
Azure PaaS ให้แพลตฟอร์มสำหรับนักพัฒนาสร้าง ติดตั้ง และจัดการแอปพลิเคชันโดยไม่ต้องกังวลเกี่ยวกับ infrastuructures Azure มีบริการกว้างขวางของบริการที่สร้างไว้ล่วงหน้า เช่น ฐานข้อมูล การส่งข้อความ และการวิเคราะห์ เพื่อช่วยให้นักพัฒนาสร้างแอปพลิเคชันได้อย่างรวดเร็ว
Azure SaaS ให้บริการแอปพลิเคชันซอฟต์แวร์แบบ fully managed บนคลาวด์ที่ผู้ใช้สามารถเข้าถึงผ่านอินเทอร์เน็ตได้ ซึ่งรวมถึงแอปพลิเคชันเช่น Office 365, Dynamics 365 และอื่น ๆ โดยรวม Azure ให้บริการคลาวด์ที่ครอบคลุมเพื่อช่วยให้ผู้ใช้สร้าง ติดตั้ง และจัดการแอปพลิเคชันของตนได้อย่างง่ายดายและมีประสิทธิภาพมากขึ้น

​Azure IaaS (infrastructure as a service)

คือบริการคอมพิวเตอร์คลาวด์ที่มีการให้บริการโดย Microsoft Azure ซึ่งมีประโยชน์ต่อลูกค้าเนื่องจากมีการให้ทรัพยากรคอมพิวเตอร์ที่เป็นเวอร์ชวลได้จากอินเทอร์เน็ต ด้วยการใช้ Azure IaaS ลูกค้าสามารถสร้างและจัดการเครื่องเสมือน วงจรเครือข่ายเสมือน และวิธีการจัดเก็บข้อมูลที่ใช้สำหรับรองรับแอพพลิเคชันและโหลดที่ต้องการ Azure IaaS ช่วยลูกค้าให้สามารถจัดการทรัพยากรได้อย่างรวดเร็วและง่ายดายตามความต้องการของธุรกิจและชำระเงินเฉพาะสิ่งที่ใช้เท่านั้น ด้วยเหตุนี้ Azure IaaS เป็นโซลูชันที่มีความคุ้มค่าสูงและเหมาะสมสำหรับ บริษัทขนาดใหญ่และเล็กขนาดเช่นกัน

ประโยชน์ของ ​Azure IaaS

  • ความยืดหยุ่น: Azure IaaS มีความสามารถในการขยายขนาดทรัพยากรคอมพิวเตอร์ได้อย่างรวดเร็วและง่ายดายตามความต้องการ โดยไม่ต้องลงทุนในการจัดการโครงสร้างกายภาพ ด้วย Azure IaaS ลูกค้ามีความสามารถในการเลือกจากหลายขนาดเครื่องเสมือนที่ได้รับการกำหนดค่าไว้และตัวเลือกการจัดเก็บข้อมูลที่แตกต่างกัน เพื่อให้ตอบสนองความต้องการของลูกค้าได้อย่างแม่นยำ
  • ความน่าเชื่อถือ: Azure IaaS มีโครงสร้างที่มีความพร้อมใช้งานสูง พร้อมกับการสร้างความ redundancy และ failover ที่ช่วยให้แอพพลิเคชันและโหลดยังคงมีการใช้งานได้แม้ว่าจะเกิดความผิดพลาดฮาร์ดแวร์หรือความขัดข้องอื่น ๆ
  • ความปลอดภัย: Azure IaaS มีคุณสมบัติและการควบคุมความปลอดภัยที่หลากหลาย เช่น การตรวจจับและป้องกันการแฮกเกอร์ การเข้ารหัสและการสร้าง firewall เพื่อป้องกันการเข้าถึงที่ไม่ได้รับอนุญาต
  • การกู้คืนภัย: Azure IaaS มีการสร้างความ redundancy และ failover ที่ช่วยให้แอพพลิเคชันและโหลดยังคงมีการใช้งานได้แม้ว่าจะเกิดความผิดพลาดฮาร์ดแวร์หรือความขัดข้องอื่น ๆ โดยมีคุณสมบัติดังนี้ เช่น Azure Site Recovery ที่ช่วยให้สามารถสำรองข้อมูลและโหลดไปยังที่อื่นได้อย่างรวดเร็ว และ Azure Backup ซึ่งเป็นบริการสำรองข้อมูลแบบออนไลน์ที่ครอบคลุมการสำรองข้อมูลของเครื่องเซิร์ฟเวอร์และแอพพลิเคชันใน Azure ด้วยการสร้างจุดกู้คืนและการกู้คืนแบบไดร์เอ็นที่มีประสิทธิภาพสูง
